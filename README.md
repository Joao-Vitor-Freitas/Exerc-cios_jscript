# Exercícios de JavaScript

Este repositório contém 5 exercícios desenvolvidos para praticar conceitos fundamentais de JavaScript, HTML e CSS.

Os exercícios abordam:

* Manipulação do DOM
* Eventos
* Estruturas condicionais
* Estruturas de repetição
* Manipulação de classes CSS
* Geolocalização
* Criação dinâmica de elementos

---

# Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript

---

# Exercício 1 — Adicionando conteúdo HTML

## Objetivo

Capturar um nome digitado pelo usuário e exibir na tela utilizando JavaScript.

## Conceitos praticados

* `window.prompt()`
* `getElementById()`
* `innerHTML`

---

# Exercício 2 — Troca de cor e texto

## Objetivo

Alterar a cor de uma div e trocar o texto dinamicamente ao clicar em botões.

## Conceitos praticados

* `addEventListener()`
* `classList.add()`
* `classList.remove()`
* `innerText`
* `innerHTML`
* Condições `if/else`

---

# Exercício 3 — Soma de valores

## Objetivo

Capturar dois números digitados pelo usuário, realizar uma soma e exibir o resultado na tela.

## Conceitos praticados

* `.value`
* `parseInt()`
* Eventos de clique
* Manipulação do DOM

---

# Exercício 4 — Criação dinâmica de elementos

## Objetivo

Adicionar bolinhas dinamicamente utilizando JavaScript.

## Conceitos praticados

* Estrutura de repetição `for`
* Variáveis de controle
* Criação dinâmica de HTML
* Classes CSS
* Arrays e cores aleatórias

---

# Exercício 5 — Geolocalização

## Objetivo

Capturar a latitude e longitude do usuário utilizando a API de geolocalização do navegador.

## Conceitos praticados

* `navigator.geolocation`
* `getCurrentPosition()`
* Manipulação dinâmica do HTML
* Eventos
